Object class
- It belongs to java.lang.Object;
- It is root of every class.
- Every class in java is child class of Object. 
- Most commonly used methods for every java class (whethere it is predefined or customized class) are defined in seperate class which is nothing but object class.
- Every class in java is child of object either directly or indirctly so that object class method are by default available to every java classe. 
- Hence object class considered as root of all java classes.
