Q4 What is the Z-index in CSS?
Ans: 
z-index specifies stack order of an element.
An element with greater stack order is always in front of element with lower stack order.
Z index only works with positioned elements and flex items
If 2 positioned elements overlap without z index specified, the element positioned last in the html code will be shown on top.
